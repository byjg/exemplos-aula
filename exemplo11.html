Exercicio 1

    Dojo:

    Criar um pagina HTML que vai criar uma calculadora que tenha as seguintes regras:

    1. Tera uma caixa de entrada
    2. Tera uma lista com os ultimos numeros digitados com as operacoes, exemplo:

        10 +
        20 +
         5 -
        25 =
         0 C

    3. Ao selecionar uma operacao, este valor sobe para a lista de ultimos numeros digitados
    4. Ao selecionar "<--" a ultima operacao deve sumir da lista e voltar para a caixa de textoo numero
    5. Operacoes basicas: +, -, /, *, '<--', Memoria Read, Memoria Write (Save), Clear, =
